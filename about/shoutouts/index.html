Shoutouts

This website is powered by Jekyll, hosted with Github Pages, and distributed by Cloudflare

Implementation

I am genuinely grateful for all the open source projects I used and advice I received from the community while building this website, including:

  • Alexey Smelyanets for providing the starting point with which I styled the Instagram button on my Social page
  • Andreas Larsen, whose elegant eased gradients generator was heavily used for many of the gradients throughout this site.
  • BigFoot for their Instapaper-like footer popups used to power my article footnotes.
  • Gapless 5 for powering the SMiLE: Heroes & Villains page's jukebox.
  • Hint for the lovely hover bubbles used throughout the site (like the Projects and Social pages).
  • jQueryScript.net for their anchor links script for displaying header anchor links on hover.
  • Michael J. Deeb for his help getting a custom domain set up with Github Pages and Hover.
  • Paul Lewis & Robert Flack, authors of Google’s Performant Parallaxing developer article, which was heavily referenced for my Largest Video Game Ever & SMiLE: Heroes & Villains pages.
  • richleland on Github for his Pygments Monokai color scheme, which I use for my code blocks.
  • shanidkv for publishing an indispensable guide explaining how to implement a dynamically fixed menu (fixed only when scrolling).

Inspiration

I drew inspiration for my website’s design from many professionals, including:

  • Bradley Taunt, a web designer & developer hybrid like myself whose original home page was the source of inspiration for my own.
  • Cezary Wojcik, a self-proclaimed keyboard wizard and iOS Developer whose projects gallery page was the initial inspiration for my own.
  • Christa Mrgan, whose work on SixColors and tasteful eye for design helped guide my own design process throughout this site.
  • Silvia Gatta, the designer behind MacStories, which inspired me to support dark mode for night reading.
  • Susan Kare, the legendary graphic designer for the original Macintosh operating system, from whose work my “Happy iMac” website logo pays homage.

Above all else I thank my Mom and Dad, my older sister Aimée, my younger brother Michael, and my loving husband Khary for their continued support as I pursue my dreams. I love you all, thanks for everything.
